1.关系运算符的优先级低于算术运算符。例如，表达式 i + j < k – 1 意思是(i + j)<(k – 1)。关系运算符都是左结合的。

2.和关系运算符一样，判等运算符也是左结合的，并且产生 0（假）或 1（真）作为结果。然而，判等运算符的优先级低于关系运算符。例如，表达式 i < j == j < k 等价于表达式(i < j)== (j < k)。如果 i < j 和 j < k 的结果同为真或同为假，那么这个表达式的结果为真。

3.C 语言虽然运算符繁多，但语句种类相对较少。根据对程序执行顺序（也就是控制流）的影响，主要可以划分为以下五大类：

核心控制流语句（三大类）
1. 选择语句 (Selection Statement)

包含：if 语句、switch 语句。

作用：相当于程序的“岔路口”，允许程序根据特定条件，在多条可选的路径中选择一条执行。

2. 重复语句 (Iteration Statement)

包含：while 语句、do 语句、for 语句。

作用：也就是我们常说的“循环”，用于支持某段代码的重复执行。

3. 跳转语句 (Jump Statement)

包含：break 语句、continue 语句、goto 语句（之前学过的 return 语句也归于此类）。

作用：打破按部就班的执行顺序，导致程序无条件地直接跳转到代码中的其他特定位置。

其他辅助语句（两类）
4. 复合语句 (Compound Statement)

作用：将多条独立的语句打包组合在一起（通常使用大括号 {}），让它们在语法上被视作“一条完整的语句”。

5. 空语句 (Null Statement)

作用：不执行任何实质性的操作（通常就是一个孤立的分号 ;），一般用于语法上需要一条语句来“占位”但逻辑上不需要做任何事的情况。

4.运算符&&和运算符||都对操作数进行“短路”计算。也就是说，这些运算符首先计算出左操作数的值，然后计算右操作数。如果表达式的值可以仅由左操作数的值推导出来，那么将不计算右操作数的值。

5.运算符!的优先级和一元正负号的优先级相同，运算符&&和运算符||的优先级低于关系运算符和判等运算符。例如，表达式 i < j && k == m 等价于表达式(i < j) && (k == m)。运算符!是右结合的，而运算符&&和运算符||都是左结合的。
